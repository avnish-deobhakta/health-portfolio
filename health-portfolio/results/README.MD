# Results Directory

## Authoritative outputs

The following files are the current source of truth for results reported in the manuscript.

- **`predictions/`** — Frozen per-image predictions for all 11 EyePACS models (2,900 images each). These are the canonical inputs to the reproduction script.
- **`csv/eyepacs_pairwise_results.csv`** — Authoritative EyePACS pairwise results, regenerated by `scripts/reproduce_results.py` from frozen predictions.
- **`csv/aptos_pairwise_results.csv`** — Authoritative APTOS pairwise summary results used in the manuscript.
- **`csv/threshold_sweep_eyepacs.csv`** — Authoritative EyePACS threshold-sweep results.
- **`csv/weight_sensitivity.csv`** — Authoritative weight-sensitivity results.

## Legacy JSON files

The JSON files below are preserved as archival outputs from the original Colab-based analysis. They are retained for completeness and for certain results not yet regenerated from frozen predictions.

- **`comprehensive_evaluation.json`** — Original combined analysis output. For pairwise manuscript results, the CSV files in `csv/` supersede this file.
- **`npj_robustness_results.json`** — Original robustness output, including bootstrap confidence intervals used in the manuscript.
- **`exhaustive_pairwise_results.json`** — Earlier legacy analysis output.
- **`full_evaluation_results.json`** — Earlier legacy analysis output.
- **`or_gate_analysis.json`** — Earlier legacy analysis output.

## Notes on numerical differences

Some legacy JSON files contain older EyePACS values that differ slightly from the authoritative CSV outputs. The authoritative manuscript values are the CSV values in `results/csv/`. These small differences do not affect any qualitative conclusion. See `REPRODUCIBILITY_MAP.md` for details.
